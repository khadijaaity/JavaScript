JS11. Arrays (MÉTODOS)

1. slice()

Selecciona una parte del array y lo retorna.

["Marta", "Juan", "Luis", "Norma", "Claudia"]



2. sort()

Ordena los elementos de un array.

["Marta", "Juan", "Luis", "Norma", "Claudia"]



3. splice()

Añade o quita elementos de un array.

["Marta", "Juan", "Luis", "Norma"]



4. toString()

Convierte el array a String.

["Marta", "Juan", "Luis", "Norma"]



5. unshift()

Añade elementos al inicio del array y retorna el nuevo tamaño.

["Marta", "Juan", "Luis", "Norma"]



6. valueOf()

Retorna el valor primitivo de un array.

["Marta", "Juan", "Luis", "Norma"]



7. concat()

Une dos o más arrays.

["Marta", "Juan"]
["Luis", "Norma", "Claudia"];



8. indexOf()

Busca un elemento en el array y retorna su posición.

["Marta", "Juan", "Luis", "Norma"]



9. join()

Junta los elementos de un array en un String.

["Marta", "Juan", "Luis", "Norma"]



10. lastIndexOf()

Busca la última aparición de un elemento en el array y retorna la posición.

["Marta", "Juan", "Luis", "Norma"]



11. pop()

Quita el último elemento del array y lo retorna.

["Marta", "Juan", "Luis", "Norma"]



12. push()

Añade un nuevo elemento al final del array y retorna su tamaño

["Marta", "Juan", "Luis", "Norma"]



13. reverse()

Invierte el orden de los elementos.

["Marta", "Juan", "Luis", "Norma"]



14. shift()

Quita el primer elemento del array y lo retorna.

["Marta", "Juan", "Luis", "Norma"]
